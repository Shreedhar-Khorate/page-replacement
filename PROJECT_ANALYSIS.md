### **🧮 src/algorithms/** - Pure JS Page Replacement Logic (4 files)
| File | Algorithm | Key Features |
|------|-----------|--------------|
| `fifo.js` | **FIFO** | Queue-based, shift oldest page |
| `lru.js` | **LRU** | Tracks last used timestamps |
| `optimal.js` | **Optimal** | Looks ahead for future use |
| `clock.js` | **Clock** | Circular list + reference bits |

**Each exports `run[ALGO](pages, frameCount)` → `{name, steps[], stats{}}`**
- **steps[]**: `{step, page, frames[], status('HIT'/'FAULT'), replacedPage}`
- **stats**: `{totalPages, hits, faults, hitRatio%, faultRatio%, frameCount}`

**Exported via `algorithms/index.js`**
